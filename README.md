Weather App

A simple and responsive Weather Application built using HTML, CSS, and JavaScript.
This app fetches real-time weather data using a weather API and displays temperature, weather conditions, humidity, and more.

 Features
	-	Displays current weather of any city
	-	Shows weather icons (sunny, cloudy, rain, mist, etc.)
	-	Handles invalid city names with a clear error message
	-	Completely built using Vanilla JavaScript
	-	Responsive design using HTML and CSS
	-	Uses API (e.g., OpenWeatherMap API)

 Tech Stack
	-	HTML5
	-	CSS
	-	JavaScript
	-	OpenWeather API
